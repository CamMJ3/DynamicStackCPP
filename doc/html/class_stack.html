◆ pop()

template<typename T>
T Stack< T >::pop ( )

Deletes the Node associated at the top of the stack.

If the stack is empty, then an error message is thrown. Otherwise, a new pointer "deleteNode" that references to the top of the stack (pointer "top"), while another variable stores the value stored in "top" to return it.

The top of the stack becomes the next Node. Finally, "deleteNode" is deleted.

Returns
The value stored in the deleted Node.

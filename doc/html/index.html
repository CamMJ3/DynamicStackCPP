Dynamic Stack in C++

Implementation of a stack in C++ using dynamic memory.

Characteristics

  • Push.
  • Pop.
  • Peek.
  • Automatic Doxygen documentation.

Project's structure

  • build/
    • Unix executable file (exec)
    • dynamStack.dSYM
  • doc/
    • html/
    • latex/
  • src/
    • stack.cpp
    • node.cpp
    • main.cpp
  • headers/
    • stack.h
    • node.h

Documentation

All the documentation was generated using Doxygen.

To view it: Go to doc/html/index.html and open with your default browser. To check all comments: Click on "Class Members" and then on "All".

Or check the PDF file "refman.pdf".

Execution

  • To run directly: Open and run the Unix executable file (exec).
  • By using commands: Open your terminal, and run...

g++ src/main.cpp -Iheaders -o dynamStack ./dynamStack

\doxysubsection{Detailed Description}
The complete program of the dynamic stack. Includes a menu function so that the user may input any desired value. 

Option 1 allows the user to input any value of type int. Option 2 allows the user to delete the \doxylink{class_node}{Node} at the top of the stack. Option 3 allows the user to view the value at the top of the stack and its size. Option 4 allows the user to exit the program.

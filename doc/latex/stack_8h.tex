\doxysubsection{Detailed Description}
Defines the structure of the \doxylink{class_stack}{Stack}. Its structure consists of a pointer called "{}top"{}. 

A stack is a linear data structure that follows the LIFO (Last-\/in, First-\/out) method, thus making the core of its functions the last element, or in this case \doxylink{class_node}{Node}, added to the stack. This file defines the constructors of the functions of the stack.
